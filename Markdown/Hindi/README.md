# नुक्ता अक्षर

क़ ख़ ग़ फ़ ज़ ड़ ढ़

नोट - किसी व्यंजन अक्षर के नीचे लगाए जाने वाले बिंदु को नुक्ता कहते हैं। क़ ख़ ग़ फ़ ज़ अन्य अक्षर उर्दू या फारसी भाषा के उच्चारण में आते हैं।
